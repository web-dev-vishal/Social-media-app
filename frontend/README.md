## 📌 Overview

This frontend delivers a modern, interactive user interface for social networking. Users can create accounts, share posts and stories, engage with content, and chat in real-time. Built with React and styled with Tailwind CSS for a seamless experience.

---

## ✨ Features

- **Authentication**: Sign up, login, OTP verification.
- **Feed**: Posts, stories, loops from followed users.
- **Content Creation**: Upload posts, stories, loops.
- **Interactions**: Like, comment, save, follow.
- **Messaging**: Real-time chat interface.
- **Notifications**: Live updates.
- **Search**: Find users and content.
- **Responsive Design**: Mobile-optimized UI.

---

## 🛠 Tech Stack

- **React**: UI framework
- **Vite**: Build tool
- **Tailwind CSS**: Styling
- **Redux Toolkit**: State management
- **Axios**: API client
- **React Router**: Navigation
- **Socket.IO Client**: Real-time features
- **React Icons**: Icon library

## ⚙️ Installation & Setup

1. Navigate to frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
